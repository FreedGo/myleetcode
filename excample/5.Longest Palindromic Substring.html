1.Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.

1.给定一个字符串s,找到最长的回文子字符串,你可以假定s最长为1000

Examples1:

			Input: "babad"

			Output: "bab"

			Note: "aba" is also a valid answer.

Examples2:

			Input: "cbbd"

			Output: "bb"

Solution

			/**
			 * Source : https://leetcode.com/problems/two-sum/

			/**
			 * 给定一个字符串,返回最长的回文字符串
			 * @param {string} str
			 * @return {string} str
			 */
			var longestPalindrome = function(s) {
				var newStr,revNewStr;
				for(var i =0 ; i < s.length ; i++){
					for(var x = 0;x <= i; x++ ){
						newStr = s.substring(x,s.length-i+x);
						revNewStr = newStr.split("").reverse().join("");
						if (newStr === revNewStr){
							return newStr;
						}
					}
				}
			};
			var s = "babad";
			console.log(longestPalindrome(s));
